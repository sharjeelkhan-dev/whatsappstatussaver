# 🔔 Daily Reminder Feature

Never miss an important WhatsApp status again! The **Daily Reminder** feature allows users to schedule notifications to check and save statuses at their preferred times.

## 🚀 Overview
The Reminder feature is built with a modern, clean UI using **Jetpack Compose**. It provides a seamless experience for managing multiple reminders with flexible options.

## ✨ Key Features
- **Custom Scheduling**: Set specific dates and times for your reminders.
- **Repeat Options**: Configure reminders to repeat daily or only once.
- **Priority Levels**: Categorize reminders by priority (High, Medium, Low).
- **Interactive Forms**: User-friendly form for adding and editing reminders.
- **Persistent Storage**: All reminders are stored locally using **Room Database**.
- **Smart Notifications**: Integrated with Android's notification system (placeholder logic).

## 🛠️ Technical Implementation
- **UI Framework**: Jetpack Compose
- **Database**: Room Persistence Library
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Injection**: Hilt
- **State Management**: Kotlin Flow & CollectAsState

## 📦 How to Use
1. Open the app and navigate to **Daily Reminder** from the Home screen.
2. Click the **+** (FAB) button to create a new reminder.
3. Enter the **Title**, **Description**, and select the **Date** and **Time**.
4. Choose the **Repeat** type and **Priority**.
5. Click **Add** to save your reminder.
6. Manage or delete existing reminders directly from the list view.
